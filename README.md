# BaskeAnaTool
Usage
=========

1. When firstly use this package, please: 
---------------------------------------
   ` source ./setup.sh`


2. Functions:
---------------------------------------
### A) sub jobs:
	BOSS jobs: <br /> 
    Hepsub -txt [-r] [path]

	Bash jobs: <br />
    Hepsub -sh [-r] [path]

    ROOT jobs: <br /> 
    Hepsub -c [-r] [path]

### B) make jobs, only the reconstruction and simulation jobs can be made
	- usage: `SimJpsi [decay.card] [number of events]`
    
    different type
    - Sim3770 
    - SimNewJpsi
    - SimJpsi
    - Sim4180
